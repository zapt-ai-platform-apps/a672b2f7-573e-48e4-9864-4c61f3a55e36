# Switching Between Light and Dark Mode

This guide explains how to toggle the app’s theme between light and dark modes.

1. **Locate the Theme Toggle:**  
   On the navigation bar or settings panel, find the Theme Toggle component.

2. **Switch Themes:**  
   Click the toggle to switch between light and dark modes.

3. **Immediate UI Update:**  
   The app will instantly update its color scheme, providing visual feedback.

4. **Persistent Settings:**  
   Your theme preference will be saved and persist on future visits.

5. **Handling Errors:**  
   If the theme does not change, check the console for error logs and try refreshing the page.
