# Setting Up the Game

## Step-by-Step Instructions

1. **Select Squad**: 
   - Choose an existing squad or create a new one from the Squad Management screen
   - Squads contain your team's roster of available players

2. **Select Match Participants**:
   - From your chosen squad, select which players will be available for this particular match
   - Toggle players in/out using checkboxes
   - Minimum 1 player must be selected

3. **Set Starting Lineup**:
   - From your match participants, select who will start on the field
   - Number of starting players depends on your team format
   - Players not selected will begin on the bench

4. **Assign Roles**:
   - Choose a goalkeeper from your starting lineup
   - Decide if goalkeeper playtime should count in totals
   - Configure substitution rules and playing time preferences

## Important Information

- **Flexible Selection**: Different match participants can be selected each game while maintaining a core squad
- **Last-Minute Changes**: Participants can be adjusted right up until game start
- **Fair Play Enforcement**: The app tracks total playtime for all participants, not just starters
- **Data Integrity**: Previous match selections don't affect future games - each match starts fresh
