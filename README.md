# Football Subs

Football Subs is a revolutionary app that elevates your football game management experience with real-time substitutions, advanced analytics, and an intuitive interface that works flawlessly across devices.

3. [Managing the Game](docs/journeys/managing-the-game.md) - Control game actions and substitutions in real-time.
4. [Reviewing Game Summary and Sharing](docs/journeys/reviewing-game-summary-and-sharing.md) - Finalize game data and share match results.
5. [Switching Between Light and Dark Mode](docs/journeys/switching-between-light-and-dark-mode.md) - Toggle the interface theme to your liking.
6. [Visualizing Players on the Pitch](docs/journeys/visualizing-players-on-the-pitch.md) - See live player positions and game dynamics.
